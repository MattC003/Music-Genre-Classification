# Music-Genre-Classification

This repository contains all materials related to my CSE 5095 (Deep Learning) course at UConn. It includes the final implementation, an earlier iteration of the code, and the final presentation and paper.

## Repository Contents

- **Final Code**  
  The completed version of the project used for the final submission.

- **Original Code Iteration**  
  An earlier version of the project showing the initial design and development approach.

- **Presentation**  
  Slides from the final project presentation.

- **Final Paper**  
  The written report describing the project motivation, methodology, results, and conclusions.

## Notes

This repository is intended to document both the **technical evolution** of the project and the **final outcomes**, providing insight into how the solution was developed and refined over time.
